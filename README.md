# Gravity Flip Connect 4 ｜ 重力翻轉四子棋

這是一款在經典四子棋基礎上，深度融合**棋盤翻轉**與**平衡先手優勢**機制的策略遊戲。

## 核心機制：重力翻轉

不同於傳統四子棋固定不變的盤面，這個遊戲的核心在於不斷變化的空間維度
*   **翻轉規則**：每一輪（雙方各落一子）結束後，棋盤將強制向右翻轉 90°。
*   **重力位移**：棋子將隨棋盤旋轉，所有存於棋盤上的棋子將遵循地心引力重新墜落與堆疊。

這項機制給傳統四子棋的固定套路帶來更多的策略變化和遊戲性，考驗玩家的思考和幾何能力，必須在落子前預判旋轉後的棋局走向。

## 平衡先手優勢

在傳統四子棋中，先手方擁有顯著的優勢。為了平衡競技性，這個遊戲引入了獨創的**補償機制**。

機制如下：
* 若先手（黑方）落子後連成四子，遊戲**並不會馬上結束**。
* 此時，後手（白方）獲得最後一子的機會，若白方能利用這最後一子同樣達成四子連線，將判定**白方獲得勝利**。
* 反之，若白方無法利用這最後一子同樣達成四子連線，將判定**黑方獲得勝利**。

## 遊戲規則

* 先手 (黑方) 落子後連成四子**且**白方的最後一子沒有連成四子，則**黑方獲得勝利**。
* 後手 (白方) 落子後連成四子，則**白方獲得勝利**。
* 若**棋盤翻轉完成**後，某一方連成四子，則**連成四子的那一方直接獲得勝利**。
* 若**棋盤翻轉完成**後，雙方都連成四子，則**平手**。
* **棋盤填滿**後，雙方都沒有達成四子，則**平手**。
